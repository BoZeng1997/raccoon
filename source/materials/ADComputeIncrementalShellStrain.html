Compute Incremental Shell Strain

Compute a small strain increment for the shell.

Description

This class computes the incremental small strain for shell elements. Please refer to ShellElements for details.

  • displacementsThe displacements appropriate for the simulation geometry and coordinate system

    C++ Type:std::vector

    Options:

    Description:The displacements appropriate for the simulation geometry and coordinate system

  • rotationsThe rotations appropriate for the simulation geometry and coordinate system

    C++ Type:std::vector

    Options:

    Description:The rotations appropriate for the simulation geometry and coordinate system

  • thicknessThickness of the shell. Can be supplied as either a number or a variable name.

    C++ Type:std::vector

    Options:

    Description:Thickness of the shell. Can be supplied as either a number or a variable name.

    C++ Type:std::string

    Options:

    Description:Quadrature order in out of plane direction
