ACGrGrMulti

Multi-phase poly-crystalline Allen-Cahn Kernel

Implements the term

for a multiphase / multigrain Grand Potential model. The parameters (hardcoded material property name mu) and (gamma_names) can be calculated using the GrandPotentialInterface material.

(mob_name) is an interfacial mobility, (variable) is the kernel variable, and (v) are the coupled phase / grain order parameters. If is a Function Material depending on any non-linear variable not listed in v those will have to be listed in args.
