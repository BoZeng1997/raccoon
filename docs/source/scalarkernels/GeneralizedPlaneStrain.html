Generalized Plane Strain Scalar Kernel

Generalized Plane Strain Scalar Kernel

The GeneralizedPlaneStrain ScalarKernel assembles components of the residual and diagonal jacobian corresponding to a given scalar variable. The values of the residual and diagonal jacobian components are fetched from the GeneralizedPlaneStrainUserObject. This object is usually set up by the GeneralizedPlaneStrainAction.
